# 终末地风格高级主题

## 边界

该主题是 QuizApp 的可选高级主题，不替换默认的跟随系统、浅色和深色主题。
主题只复用通用的视觉规律，不包含《明日方舟：终末地》或鹰角启动器的图片、
Logo、字体文件、图标、音效和其他美术资产。

## 本地参考

参考来自本机鹰角启动器 1.5.0 的生产 Web 构建：

- 冷黑主背景和分层的近黑表面；
- 黄色主操作、焦点和进度状态；
- 白色文本按透明度建立信息层级；
- 珊瑚红仅用于错误和警示；
- 2-4px 小圆角、细边框和紧凑工具条；
- 交互状态主要使用 150-300ms 的淡入与短距离位移。

游戏本体是 Unity IL2CPP 编译产物，不存在可直接阅读的 C# 工程源码。本主题不反编译
游戏业务逻辑，也不从资源包提取或分发专有素材。

## 开源参考

辅助参考 [cloud-oc/endspace](https://github.com/cloud-oc/endspace) 的 `theme`
分支。该项目采用 Apache-2.0 许可证，是受终末地官网启发的 NotionNext 主题。本项目
只参考以下通用做法，不复制其页面、品牌文案或媒体素材：

- 桌面侧栏和移动导航使用不同的信息密度；
- 活动位置由窄指示条和单一强调色表达；
- 主字体承载正文，等宽字体只用于状态、编号和元信息；
- 卡片默认保持克制，仅在焦点和操作状态使用黄色；
- 进度动效与真实加载状态绑定，不使用虚假循环进度。

## QuizApp 映射

- 应用壳：冷黑底色、黄色品牌标记、平面导航层级；
- 首页和学习中心：摘要块使用细直线分层，数值保持高对比；
- 题库：目录卡片和模式入口使用紧凑边框，不使用装饰图片；
- 练习与复习：正确答案使用黄色边框，错误和错题集使用珊瑚红；
- 手写：工具栏和画布明确分层，选中工具使用黄色状态；
- 动效：继续遵守“减少界面动画”设置，不新增循环装饰动画。
