# QuizApp Native v2.0.0-alpha.2

This prerelease installs beside the stable WebView APK as `com.quizapp.preview`.
It does not replace `v1.0.18` and does not yet migrate legacy application data.

## Added

- Android root shared storage at `/storage/emulated/0/QuizApp` with separate
  `QuestionBanks`, `Backups`, `Exports`, `Notes` and `RecycleBin` folders.
- Android 11+ all-files-access permission flow and external file-manager entry.
- Startup and foreground bank synchronization using relative hierarchy,
  file metadata and SHA-256 fingerprints.
- In-app shared-storage file tree with sync states, folder creation, JSON
  import, recycle, restore and permanent-delete actions.
- All six `v1.0.18` color themes as independent native presets, plus the
  isolated Endfield advanced theme.
- Persistent 0-18 px global component corner-radius control.
- Immutable `v1.0.18` feature/theme migration matrix.

## Data behavior

- Folder and filename determine the subject/chapter/nested hierarchy.
- Removing a managed JSON hides its bank without immediately deleting learning
  records. Restoring the file reactivates the bank.
- The preview package keeps its private SQLite database separate from the
  stable APK. Legacy migration is still incomplete.

## Known limits

- No Android hardware/device run was available for this build; Android storage
  permission and system file-manager behavior are build-verified but still need
  phone/tablet device acceptance.
- Features listed as incomplete in
  `native/docs/legacy-feature-migration-matrix.md` are not yet migration-complete.
