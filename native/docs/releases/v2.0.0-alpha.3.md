# QuizApp Native v2.0.0-alpha.3

This prerelease installs beside the stable WebView APK as `com.quizapp.preview`.
It does not replace `v1.0.18` and does not yet migrate legacy application data.

## Added

- The validated public postgraduate bundle is now named `27考研题库包`.
- Android builds can package the bundle as private first-launch assets without
  committing the generated SQLite database or roughly 142 MB of media.
- First launch stages and validates the complete bundle before installation.
- Fresh databases and existing empty alpha databases receive the default banks.
- Existing banks, practice sessions, wrong-book entries, review data, notes,
  statistics, AI records and settings prevent automatic replacement.
- Empty databases are backed up before replacement and failed copies roll back.

## Bundle identity

- Display name: `27考研题库包`
- Internal source provider: `xiaoyivip` (retained for update compatibility)
- Questions: 870
- Sections: 72
- Media blobs: 1936

## Compatibility baseline

- All six `v1.0.18` palettes remain available.
- System, light and dark appearance modes remain independent from palettes.
- Endfield remains an additional advanced theme.
- The persistent global component radius remains adjustable from 0 to 18 px.

## Known limits

- No Android hardware/device run was available for this build. APK asset and
  manifest verification do not replace phone/tablet acceptance.
- Legacy `v1.0.18` application-data migration is still incomplete, so the
  preview package remains separate from the stable package.
